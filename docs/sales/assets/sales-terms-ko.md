# 판매/설명 용어 표

| 용어 | 쉬운 설명 | 정확한 설명 |
| --- | --- | --- |
| Trading Decision AI | 매매 방향을 제안하는 AI | `TradeDecision` 스키마를 반환하는 의사결정 계층 |
| Chief Review AI | 운영 요약을 해주는 AI | decision, risk, health, alert를 종합해 운영 모드를 제안 |
| Risk Engine | 실행을 통제하는 안전장치 | leverage, risk, slippage, stale data 등을 검사하는 정책 계층 |
| Paper Trading | 실제 돈이 아닌 시뮬레이션 매매 | 주문/체결/포지션/PnL을 기록하는 모의 실행 모드 |
| Replay | 과거 시나리오 재현 | decision/risk/execution 흐름을 다시 돌려보는 검증 방식 |
| Audit Log | 추적 기록 | agent, risk, execution, alert 이벤트를 저장하는 이력 |
