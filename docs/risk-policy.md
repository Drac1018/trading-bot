# Risk Policy

## 강제 규칙

- 최대 레버리지: `3x`
- 거래당 최대 리스크: `1.00%`
- 일일 손실 한도: `2.00%`
- 연속 손실 `3회` 이후 신규 진입 선호 모드: `HOLD`
- 손절/익절 누락 또는 방향 오류 시 진입 차단
- stale / incomplete market data 차단
- 슬리피지 임계치 초과 시 주문 거부
- 스키마 검증 실패 시 결정 폐기
- 리스크 정책과 AI 충돌 시 정책 우선
- 수동 정지 스위치 존재

## 모드 규칙

- 기본 모드: `paper`
- `live`는 기본 비활성
- 향후 `live`는 환경 플래그 + 수동 승인 + 리스크 승인 + 감사 로그가 모두 필요
